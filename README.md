# Simple Habit Tracker

A minimal macOS habit tracker built with SwiftUI.

## Features

- Add new habits (Button or Command+N)
- Check off habits daily (Resets visually each day, keeps history)
- Delete habits (Swipe or context menu)
- Persists data automatically

## How to Build and Run

1. Open Terminal in this folder.
2. Run the build script to create the App:
   ```bash
   ./build_app.sh
   ```
3. Look for `SimpleHabitTracker.app` in the folder.
4. Drag `SimpleHabitTracker.app` to your Applications folder or Dock.
5. Launch it!

## Development

To edit the code, open `Package.swift` in Xcode.
